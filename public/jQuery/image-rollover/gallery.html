Image Rollover Design Pattern

User Story

As a user, I want to see another view of each image. For example when visiting a city what is the daytime and evening view?

Dev Steps

As a dev, use jQuery to apply image rollover interaction: when mouse is hovering over the image then reveal the secondary image.

  1. One image
  2. Many images - refactor code to reusable functions

Technologies

  • jQuery
  • JavaScript
  • HTML
  • CSS
