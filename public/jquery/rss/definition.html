AJAX

JavaScript AJAX cannot directly access XML content when hosted on an external domain name. The browser will restrict the XSS (Cross-site scripting) to prevent phishing from predetor web sites.

The workaround to the browser security is to proxy the external content into our domain name. I am providing a Node.js proxy that will screen scrape the RSS content and mirror it locally.
