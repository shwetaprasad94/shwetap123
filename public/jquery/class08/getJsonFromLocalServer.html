Goal

  • As a user, I want to buy fruit from my local grocer online.
    As a dev, output the fruit using AJAX to get JSON from the local app server (Node.js)
  • As a user, I want to see the fruit names with the background colour representing the fruit, so I may buy from my local grocer online.
    As a dev, output the fruit keys with the colour as the background fill.

Technologies

  • jQuery
  • jQuery AJAX
  • JSON
  • JavaScript
  • node.js
