Goal

  1. As a user, I want to buy fruit from my local grocer online.
    As a dev, output the fruit using AJAX to get JSON from the local app server (Node.js)
  2. As a user, I want to see the fruit names with the background colour representing the fruit, so I may buy from my local grocer online.
    As a dev, output the fruit keys with the colour as the background fill.
  3. As a user, I want find the best price for apples, even if the database query takes a long time.
    As a dev, I want to apply a design pattern to inform the user of a slow simulated database query.

Technologies

  • jQuery
  • jQuery AJAX
  • JSON
  • JavaScript
  • node.js
