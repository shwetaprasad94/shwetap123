Photography

Photos will be hosted by Flickr. We will leverage the tools provided by Flickr especially input and image manipulation services. Apps are available for mobile devices for convient photo uploading.

Goal

Our client has used the Flickr iOS app to upload photos of their YVR store to Flickr which are available to the public.

As a user, I want to see photos of the YVR store

  1. Using jQuery AJAX, connect to Flickr to request photos
  2. Copy and Paste one JPG path, and use jQuery to add one photo to page
  3. Dynamically read the JPG path, and use jQuery to add one photo to page
  4. Loop through Flickr response and display all photos to page

As a user, I want to see 100 photos of the YVR store

  1. Create a server route with JSON containing 100 photos; switch from Flickr feed to Flickr API
  2. Change server JSON to return JPG path

Technologies
