Shared HTML menu

Goal

  • As a user, I want to navigate across the entire site with a menu
  • As a dev, build an HTML menu snippet that will be shared across many web pages. HTML cannot be altered!
  1. As a dev, setup the file system for my JavaScript file
  2. As a dev, request menu and output to console
  3. As a dev, append HTML content to body
  4. As a dev, insert HTML content to container in final location
  5. As a dev, apply CSS to HTML content
  6. As a user, when I click on the City menu item, I want a sub menu to appear with city names
  7. As a dev, I want to listen to the click event and console log "clicked"
  8. As a dev, request city JSON and output to console
  9. As a dev, insert city JSON content to container in final location

Requirements change

After client demo, the sub menu cities should display a background colour based on the weather condition. For example ⁠⁠Overcast is light blue, ⁠⁠Cloudy is light grey, and ⁠Mostly cloudy⁠⁠ is dark grey

  1. As a dev, I want to console log the weather conditions.
  2. As a dev, I want to transform the weather condition to a CSS class name.
  3. As a dev, define the CSS class name.
