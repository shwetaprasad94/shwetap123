Shared HTML menu

Goal

  • As a user, I want to navigate across the entire site with a menu
  • As a dev, build an HTML menu snippet that will be shared across many web pages.
  1. As a dev, request menu and output to console
  2. As a dev, append HTML content to body
  3. As a dev, insert HTML content to container in final location
  4. As a dev, apply CSS to HTML content
  5. As a user, when I click on the City menu item, I want a sub menu to appear with city names
  6. As a dev, I want to listen to the click event and console log "clicked"
  7. As a dev, request city JSON and output to console
  8. As a dev, insert city JSON content to container in final location

Technologies

  • JavaScript jQuery
  • JavaScript jQuery AJAX
  • HTML
  • CSS
