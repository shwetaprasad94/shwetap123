MeCard format

Is similar to the vCard contact standard, but is less verbose due to the simple datum structure.

MECARD:
N:Last,First;
NOTE:Occupation title;
TEL:16041234567;
EMAIL:email@address.ca;
URL:http://www.website.com;;

Goal

As a user, I want a MeCard generator.

As a dev, get these fields from a HTML form: first & last name, job title, and web address. On submit display the captured data in a MeCard format.

Dev steps

  1. Bind submit (event) to form (element)
  2. cancel form action event.preventDefault() or return false;
  3. then update textarea with "hello"
  4. change the textarea to display first name value
  5. change the textarea to display all field's value
  6. change the textarea to display all field's value in MeCard format

Technologies

  • jQuery
  • JavaScript
  • HTML
