MeCard format

Is similar to the vCard contact standard, but is less verbose due to the simple datum structure.

MECARD:
N:Last,First;
NOTE:Occupation title;
TEL:16041234567;
EMAIL:email@address.ca;
URL:http://www.website.com;;

Goal

    • As a user, I want a MeCard generator.
    • As a dev, get these fields from a HTML form: given and family name, job title, and web address. On submit display the captured data in a MeCard format.
    • As a user, I want a QR code storing a MeCard.
    • As a dev, get these fields from a HTML form: given and family name, job title, and web address. On submit display the captured data as a QR Code that visualizes the MeCard format.

Dev incremental steps

  1. Listen to submit button click event
  2. Cancel form action event.preventDefault() or return false;
  3. Then update textarea with "hello"
  4. Change the textarea to display first name value
  5. Change the textarea to display all field's value
  6. Change the textarea to display all field's value in MeCard format

After the MeCard is generated and stored in the textarea form field

  1. Listen to button click event
  2. Update image element source attribute with QR Code path with textarea value as API parameter
