Twitter

API

Goal

As a user, I want to read the latest news of client VanArts.

Acceptance criterion

  • Display the tweet text with date in this date format: mm/dd/yyyy, HH:MM:SS

Developer steps

  1. Find endpoint in documentation
  2. Research Twitter API client module
  3. Create a Twitter App
  4. Generate and reference credentials in application config (i.e. credentials.json)
  5. Create a hapi route and output JSON using twit module

Technologies

  • Twitter JS API
  • Twitter API client for node
  • hapi.js
  • node.js
  • jQuery AJAX
  • jQuery
  • JavaScript
