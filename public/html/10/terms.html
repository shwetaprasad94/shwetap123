Terms

Plugins

Easy to implement with limited customization

Example

Facebox social plugins. We added a Like plugin (iframe), with only three layout options. Has access to public info only (Friends list).

API

API is the direct library that accesses a web site. Rich customization but more difficult to learn. Private content is accessible thru authentication.

Example

Facebook API. We can read more private information after user grants permission

Abstraction layer

Is a broad library that accesses multiple services or sites. Similar learning curve to API and slightly less customization

Example

Mapstraction: Allows us to change mapping providers without altering our code. The library has more general features common across all the mapping providers.
