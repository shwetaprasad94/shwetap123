NPM

Node Package Manager

npm install look for depencies in package.json and install when missing

npm update look for depencies in package.json but it will get the latest

node_modules folder stores all the open-source dependencies

Setup

npm init will setup your package.json with a step-by-step wizard

version <major>.<minor>.<patch>. The difference between major versions is a breaking change. Meaning the developer will likely be required to change their code to use the new version. Minor version is safe to consume as there are no breaking changes, likely new features are added. For example the current version of jQuery is v2.1.4, when v2.2.0 is released we can all upgrade to this version without changing our code. Though we should test our web pages to ensure functionality. Patch version will contain bug fixes and security updates.
